# Simple Budget Tracker API

## Project Overview
The Simple Budget Tracker API allows users to record daily expenses, categorize them, and view weekly or monthly spending summaries.  
It is a backend REST API built with Django and Django REST Framework, focused on secure, user-specific expense tracking.

---

## Tech Stack
- Python
- Django
- Django REST Framework
- SQLite (development)
